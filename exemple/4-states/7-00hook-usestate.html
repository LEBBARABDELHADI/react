Chap 04 State

Hooks

Les Hooks sont une nouveauté de React 16.8.
Dans cet exemple, On appelle 2 setState successif.
On pourrait s'attendre à avoir dans la console
Valeur initiale > Nouvelle valeur > Nouvelle valeur modifiée
mais nous avons bien uniquement
Valeur initiale > Nouvelle valeur modifiée.
Car React a groupé les setState successif pour ne faire qu'un seul re-render.
