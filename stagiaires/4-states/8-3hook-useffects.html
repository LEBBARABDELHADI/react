Hooks

Les Hooks sont une nouveauté de React 16.8.
useEffect. Celui-ci prend deux paramètres :
Une fonction dont le rôle est d’appliquer les effets attendus ;
Cette fonction peut retourner une autre fonction dont le rôle sera de nettoyer
(typiquement le disconnect après le connect) ; Un tableau des dépendances (qui fonctionne exactement sur le même modèle que useCallback et useMemo)
pour n’exécuter l’effet que si une des dépendances a changé de valeur ;
